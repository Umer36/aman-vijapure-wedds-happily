## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:8080](http://localhost:8080) to view the schedule.

## Customization

Edit the following in `src/pages/Index.tsx`:
- Wedding dates and events
- Venue information
- Custom messages
